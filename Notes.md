Create Virtual env
python3 -m venv venv

Activate the virtual environment
macOS / Linux
source venv/bin/activate

Windows (PowerShell)
venv\Scripts\Activate.ps1

Windows (CMD)
venv\Scripts\activate


Once activated, your prompt changes to something like:

(venv) $
